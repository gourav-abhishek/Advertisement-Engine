# Ad Targeting & Real-Time Bidding Engine Web App

## Project Overview
This project is a comprehensive Ad Targeting and Real-Time Bidding (RTB) Engine web application designed to simulate a production-grade programmatic advertising platform. It incorporates a machine learning model  to predict bid outcomes, optimize bidding, and improve ad targeting efficiency.

## Features
- **Machine Learning Model**: Predicts win probability, click-through rate (CTR), and recommends optimal bids based on user inputs like device, ad format, audience, time, and location.
- **Real-Time Bidding**: Live auction simulation with bid input, countdown timer, and instant result display.
- **Targeting Options**: Device type, geographic location, audience segments, ad format, and time-of-day targeting.
- **Performance Metrics**: Tracks impressions, clicks, conversions, win rates, CPM, and ROI with interactive dashboard and visual analytics.
- **Interactive Elements**: Real-time bid feed, dynamic charts, and feedback for user interaction.
